# 当季水果蔬菜微信小程序包

这是从当前 Vue/Vite Web 项目整理出的轻量微信小程序原生包，可直接用微信开发者工具导入 `apps/miniprogram` 目录预览。

## 导入方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本目录：`apps/miniprogram`。
4. AppID 可先使用测试号或在 `project.config.json` 里替换为正式小程序 AppID。

## 当前包含

- 当季：月份筛选、搜索、水果/蔬菜筛选、当季食材卡片。
- 分类：按食材子类展示代表食材。
- 产地：省份选择与代表蔬果展示。
- 排行：水果/蔬菜营养成份排行。
- 本地内置 82 条食材数据，不依赖本地 Node API。

## 说明

微信小程序主包体积限制较严格，原 Web 项目的图片资源约 206MB，未直接打入小程序包。当前小程序包使用轻量本地数据和占位式视觉。后续若有 CDN/对象存储域名，可以把图片字段改为远程资源加载。

Web 版里的 ECharts 中国地图、GSAP 动画和复杂详情交互没有直接移植到主包中；如果要完整复刻 Web 版体验，建议后续用分包、云开发或远程图片/CDN 继续拆分。
